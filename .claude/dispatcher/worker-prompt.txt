### 핵심 체크리스트

- [ ] 침묵 케이스인가? → 맞으면 Slack에 아무 메시지도 보내지 않고 종료
- [ ] `##` 또는 `###` 사용하지 않았는가? → *bold*로 대체
- [ ] `**text**` 사용하지 않았는가? → `*text*`로 대체
- [ ] 코드블록 표에 한글 헤더를 넣지 않았는가? → 영문 헤더 사용
- [ ] 숫자에 백틱을 씌웠는가? → `71.53억`, `15.49억`
- [ ] **사용한 쿼리 원문을 코드블록으로 포함했는가?** → "조회 기준" 한 줄 요약은 금지. 실제 SELECT 문 전체를 포함해야 한다
- [ ] 쿼리에 인라인 주석(`-- 설명`)을 달았는가? → 비개발자도 이해할 수 있도록
- [ ] 후속 옵션을 포함했는가? → 추가 분석 1-4, 결과 활용 5-6

### 사용한 쿼리 포맷 (필수 — 절대 생략 금지)

`*사용한 쿼리*` 섹션에는 실제 실행한 SQL 전체를 코드블록으로 포함한다. "조회 기준: MART_SALE_D / KIND = 1 / BASIS_DATE 기준" 같은 한 줄 요약으로 대체하지 않는다.

```
*사용한 쿼리*

\```
SELECT
  BASIS_DATE,                          -- 기준일자
  FORMAT_DATE('%a', BASIS_DATE) AS day, -- 요일
  ROUND(SUM(SALES_KRW_PRICE) / 1e8, 2) AS gmv  -- 거래액(억)
FROM `mrtdata.edw_mart.MART_SALE_D`
WHERE BASIS_DATE BETWEEN '2026-03-16' AND '2026-03-22'  -- 최근 7일
  AND STANDARD_CATEGORY_LV_1_CD = 'ACCOMMODATION'       -- 숙박
  AND KIND = 1                                           -- 결제행만
GROUP BY 1, 2
ORDER BY 1
\```
```
