# 포인트용 파일 추출

예약자 정보 Excel(파일 A)을 입력받아, 수량 기준으로 행을 확장하고 포인트 지급에 필요한 3개 컬럼만 추출하여 새 Excel(파일 B)을 생성한다.

## 워크플로

1. 사용자에게 입력 파일 경로를 확인한다 (이미 제시했으면 그대로 사용)
2. `scripts/extract_points.py`를 실행한다
3. 결과 파일 경로와 행 수를 보고한다

## 입출력 사양

### 입력 (파일 A)
- 형식: `.xlsx`
- 필수 컬럼: `예약번호`, `여행자 연락처`, `수량`
- 첫 번째 행은 헤더

### 출력 (파일 B)
- 형식: `.xlsx`
- 컬럼 3개:

| 출력 컬럼 | 원본 컬럼 | 설명 |
|-----------|-----------|------|
| `recipient_no` | 여행자 연락처 | 전화번호 (문자열) |
| `reservationNoWithCount` | 예약번호 | 수량만큼 동일 행 반복 |
| `voucher1` | — | 빈 칸 (사용자가 나중에 난수코드 수기 입력) |

### 행 확장 규칙
수량 N이면 동일 예약번호·연락처 조합으로 N행을 생성한다.

**예시**: 수량=3인 예약 1건 → 출력 3행
```
recipient_no  | reservationNoWithCount      | voucher1
01012345678   | TNA-20260413-00001133       |
01012345678   | TNA-20260413-00001133       |
01012345678   | TNA-20260413-00001133       |
```

## 실행 방법

```bash
python <skill-path>/scripts/extract_points.py <입력파일> [--output <출력파일>]
```

- `--output` 미지정 시: 입력 파일과 같은 디렉토리에 `point_output_YYYYMMDD.xlsx`로 자동 생성
- 동일 이름 파일이 이미 있으면 `_v2`, `_v3` suffix를 붙여 새로 생성 (덮어쓰기 금지)

## 전화번호 처리
- Excel에서 숫자로 저장된 경우 문자열로 변환한다
- 10자리이고 `10`으로 시작하면 앞에 `0`을 추가한다
- 11자리 010 번호는 하이픈 포맷 적용: `010-XXXX-XXXX`
- 예: `1012345678` → `010-1234-5678`

## 에러 처리
- 필수 컬럼이 없으면 어떤 컬럼이 누락됐는지 명시하고 중단
- 수량이 비어있거나 0이면 해당 행 건너뛰고 경고 출력
- 입력 파일이 없으면 경로 재확인 요청
